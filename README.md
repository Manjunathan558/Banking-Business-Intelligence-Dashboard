# Banking Business Intelligence Dashboard

## Overview

This project presents an interactive Banking Business Intelligence Dashboard developed to analyze key banking metrics and generate actionable business insights. The dashboard enables users to monitor financial performance, customer behavior, loan distribution, deposit trends, and operational KPIs through dynamic visualizations.

## Objectives

* Analyze banking loan and deposit performance.
* Monitor key financial KPIs using interactive dashboards.
* Identify customer trends across income bands and nationalities.
* Support data-driven decision-making with meaningful business insights.

## Tools & Technologies

* **Power BI** – Interactive dashboard development and data visualization
* **SQL** – Data extraction and analytical queries
* **Python (Pandas, NumPy)** – Data cleaning and preprocessing
* **Excel** – Initial data preparation

## Key Features

* Executive KPI Dashboard
* Loan Analysis
* Deposit Analysis
* Customer Segmentation
* Income Band Analysis
* Nationality-wise Banking Analysis
* Financial Performance Overview
* Interactive Filters and Slicers

## Key Insights

* Identified loan distribution across different customer income groups.
* Analyzed deposit patterns and customer account performance.
* Compared banking activities across multiple nationalities.
* Tracked important financial KPIs, including total loans, deposits, fees, and customer accounts.

## Skills Demonstrated

* Data Cleaning
* Exploratory Data Analysis (EDA)
* SQL Querying
* Business Intelligence
* Data Visualization
* Dashboard Design
* KPI Development
* Business Insight Generation

## Outcome

The dashboard provides a comprehensive view of banking operations, helping stakeholders monitor performance, identify trends, and make informed business decisions through interactive visual analytics.
